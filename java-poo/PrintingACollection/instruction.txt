O modelo de exercício possui uma classe SimpleCollection predefinida, que é usada para representar um grupo de valores. 
A classe não possui o método toString usado para impressão.

Implemente um toString método para a classe que terá o desempenho demonstrado nos exemplos a seguir.

Sample output
The collection alphabet is empty.

The collection alphabet has 1 element:
a

The collection alphabet has 2 elements:
a
b

The collection alphabet has 3 elements:
a
b
c

Sample output
The collection characters is empty.

The collection characters has 1 element:
magneto

The collection characters has 2 elements:
magneto
mystique

The collection characters has 3 elements:
magneto
mystique
phoenix
